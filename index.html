Jiajun Wang's Introduction

Jiajun Wang is a Master of Computer Science student at Northeastern University, with a strong foundation in software engineering and game design. He earned a dual bachelor's degree in Computer Science from the University of Michigan and Electrical and Computer Engineering from Shanghai Jiao Tong University.

Jiajun is a full-stack software engineer with experience in both frontend and backend development. Fake Stack Overflow

This project is a **full-stack web application** inspired by Stack Overflow, designed to allow users to post, answer, and interact with programming-related questions. Built using TypeScript, Node.js, Express, and MangoDB, this platform features a responsive UI, user authentication, and dynamic content updates. Users can upvote or downvote answers, search for relevant questions, and track their contributions.

Slime Handed

Made as the final class project for EECS 494: Intro to Game Development. The development occurred over a month and a half with playtesting occuring each week. Steganography and Modern Communication

By combining the intricate arts of steganography and cryptography within an Android application called Cryptex. Cryptex is a product aiming to provide an all-in-one communication platform integrated with steganography, which is secure, fun, and fast. Cryptex is designed to be two parts, which are the sender and receiver separately. Sender allows users to retrieve keys from the receiver, input text, choose encryption media and send encrypted while the receiver can receive text and decrypt text. Also, we add a public key that is only known between the sender and the receiver to ensure safety.
